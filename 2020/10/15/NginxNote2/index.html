Nginx学习笔记（二）

在上一篇文章说了我对Nginx理解，这篇文章就来说说Nginx的在Linux下的安装和基础的配置。

Nginx安装

我这里写下Linux下的安装过程。

Linux（Centos7）

1. 安装依赖包

// 一键安装依赖包
yum -y install gcc zlib zlib-devel pcre-devel openssl openssl-devel

2. 下载并解压后安装包

cd /usr/local
// 创建文件夹
mkdir nginx
// 下载安装包(具体版本根据自己需求)
wget http://nginx.org/download/nginx-xx.xx.xx.tar.gz
// 解压
tar -xvf nginx-xx.xx.xx.tar.gz

3. 安装Nginx

// 进入Nginx目录
cd /usr/local/nginx/nginx-xx.xx.xx
// 执行
./configure --prefix=/usr/local/nginx --with-http_stub_status_module --with-http_ssl_module
// 执行make命令
make
// 执行make install命令
make install

4. 启动Nginx

// 进入nginx启动目录
cd /usr/local/nginx/sbin
// 运行nginx
./nginx

5. 将Nginx添加到全局变量中

添加到全局变量中后，就可以在任何地方使用Nginx命令啦

ln -s 你的nginx的位置 /usr/local/bin/
//ln –s 源文件 目标文件

6. Nginx常用命令

// 启动
nginx
// 关闭
nginx -s stop
// 重载配置文件
nginx -s reload
// 检查nginx语法，建议每次在重载前检查
nginx -t

总的来说，安装还是蛮简单的。

配置Nginx

静态服务器配置

server {
  listen 80;   
  server_name  localhost;    
  client_max_body_size 1024M;  # 控制请求内容大小
    
  location / {   
   root   xxxxx;  # 静态文件地址   
   index  index.html;  # 静态文件类型
  }    
}

反向代理配置

server 
{
  listen     80;                                                         
  server_name  访问地址/域名;
  client_max_body_size 1024M;  # 控制请求内容大小

  location / {
    proxy_pass 代理地址及端口;
    proxy_set_header Host $host:$server_port;
  }
}

总结

这是我第一次进行Nginx的实践，只是对其进行了初步的了解和使用，如果有写的不对或写的不好的地方，请见谅。
